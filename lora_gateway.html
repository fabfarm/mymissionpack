LoRa Gateway Bridge Documentation

The LoRa Gateway Bridge facilitates communication between LoRaWAN devices and the ChirpStack network server. Typically, it is configured via ChirpStack or a similar platform.

Host: 192.168.1.243

Port: 1700

Default Username: Not required
